The Xcode Keyboard Shortcuts

This is the official home of the Xcode Keyboard Shortcuts as published by me Colin Wheeler (aka Cocoa Samurai.) I created this repository to act as a central location from which to keep track of updates to the shortcuts and allow people to see the forks of this guide.

How do I contribute

Honestly the easiest way you can contribute a version is to fork this project and do a pull request. I prefer this way because then you get properly credited as a contributor.
